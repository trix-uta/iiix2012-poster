UtaProxy is a suite of tools for studying within-document navigation

  • UtaProxy (a fork of UsaProxy 2.0 by Richard Atterer & Monika Wnuk))
  • UtaProxy Report Generator — for generating interactive visualizations based on logged interaction data
  • UtaProxy Log2DB — for storing UtaProxy logs into a database

Usage examples

  • Session report of web document browsing (see also filtered log)
